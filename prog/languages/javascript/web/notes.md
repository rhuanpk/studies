# JavaScript Notes

## Conventiosn

- **Vaiable naming**:
	camelCase

## General

- **Variable declarations**:
	`let variableName`

- **Quotation marks**:
	bot `""` and `''` are to write raw strings.

- **String concatenate**:
	`'text ' + variable`

- **Template strings**:
	<code>\``text ${variable}`\`</code>

## Built-in

- Browser notification:
	`alter()`

- Browser input:
	`prompt()`

- Console printing:
	`console.log()`

## Tips & Tricks

- `let` with `prompt`:
	`let input = prompt()`

## Conditionals

Traditional `if`:
```js
if (<expression>) {
	<commands>
} [else {
	<commands>
}]
```

Only _true block_ `if` shorthand:
```sh
if (<expression>)
	<commands>
```
